# 电商平台首页改版 A/B Test 增长实验分析

本项目基于 Jupyter Notebook，围绕电商平台首页改版场景，使用 A/B Test 方法评估新版首页对用户点击、购买转化及业务价值的影响，完整展示了从业务问题拆解、指标设计、实验分析到结论输出的全过程。

## 项目亮点
- 聚焦真实业务场景，体现数据分析在增长决策中的实际应用
- 采用 A/B Test 分析框架，具备完整的实验评估思路
- 围绕 CTR、CVR、Revenue 等核心指标展开，兼顾用户行为与商业结果
- 结合用户类型、设备、渠道等维度进行分群分析，提升业务洞察深度
- 输出可落地的业务建议，体现从数据分析到业务决策的闭环能力

## 项目背景
某电商平台计划对首页进行改版，希望通过优化商品推荐布局、视觉设计和内容展示方式，提升用户点击率与购买转化率。

为了验证新版首页是否优于旧版首页，本项目基于 A/B Test 实验思路，将用户随机分配至 A 组（旧版首页）和 B 组（新版首页），并对两组在关键指标上的表现进行对比分析，以评估页面改版的实际效果。

## 分析目标
本项目重点回答以下问题：
- 新版首页是否显著提升了用户点击率（CTR）
- 新版首页是否显著提升了用户购买转化率（CVR）
- 新版首页是否带来了更高的收入或商业价值
- 哪些用户群体对新版首页的响应更明显
- 是否建议进一步扩大新版首页的流量覆盖范围

## 数据来源
本项目使用模拟生成数据，用于复现电商首页改版场景下的 A/B Test 分析流程。
数据集特征包括：
- 样本量约为 200,000 用户
- 用户类型包括新用户与老用户
- 实验组分为 A 组（旧版首页）与 B 组（新版首页）
- 用户按 50% / 50% 随机分配至不同实验组
- 字段包括：`user_id`、`user_type`、`group`、`device`、`channel`、`click`、`order`、`revenue`、`event_date`

该数据主要用于模拟不同页面版本对点击、转化与收入表现的影响，从而完成完整的实验评估流程。

## 分析内容

### 1. 数据读取与基础检查
对实验数据进行加载、字段查看和描述性统计分析，确保数据结构清晰、分析基础可靠。

### 2. 核心指标分析
对 A/B 两组的 CTR、CVR、Revenue 等核心指标进行对比，观察新版首页是否带来整体提升。

### 3. 显著性检验
使用统计检验方法验证实验组与对照组在关键指标上的差异是否具有统计显著性，确保分析结论具备可信度。

### 4. 用户分群分析
从用户类型、设备类型、渠道来源等维度拆解实验结果，识别不同群体对新版首页的响应差异。

### 5. 业务解读与建议
结合实验结果，从业务角度判断新版首页是否值得推广，并提出更具针对性的优化建议。

## 主要结论

根据项目分析结果，可以得到以下结论：
- 新版首页在整体上带来了 CTR 和 CVR 的提升，说明页面改版对用户点击和购买转化具有积极影响
- 关键指标的 A/B Test 结果达到统计显著，说明实验结论具有较强可信度
- 新版首页对不同用户群体的影响存在明显差异，其中部分高意图用户和新用户群体提升更为显著
- 不同设备与渠道下的实验表现并不完全一致，说明后续仍可结合具体场景做精细化优化
- 从业务价值角度看，新版首页具备逐步扩大流量覆盖的潜力，适合分阶段推进上线策略
